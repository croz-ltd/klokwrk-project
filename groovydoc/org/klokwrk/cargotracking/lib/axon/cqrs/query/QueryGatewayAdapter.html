        • <R, Q> @SuppressWarnings(value: GrUnnecessaryPublicModifier)
          R query(Q query, Map<String, ?> metaData, Class<R> queryResponseClass)

          Delegates calls to the QueryGateway.query() method.

          In case when an exception is thrown from QueryGateway, it unwraps details exception (if available), and rethrows it to the caller.

          throws:
          AssertionError when query is null.
          throws:
          CompletionException when cause is not an instance of QueryExecutionException.
          throws:
          QueryExecutionException when details exception is not available.
          throws:
          Throwable when available as details of QueryExecutionException.
          Parameters:
          query - The query to be executed.
          metaData - The metadata to dispatch with the query.
          Type Parameters:
          R - The type of result expected from query execution.
          Returns:
          the result of query execution.
