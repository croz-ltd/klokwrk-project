        • MetaDataAssertion isSuccessful()

          Weather metadata contains all properties as returned from web controller.

          Those properties differ from ones returned from application facade API. For example, web interceptors will add metaData.http or metaData.general.locale properties.

        • MetaDataAssertion isSuccessful_asReturnedFromFacadeApi()

          Weather metadata contains basic properties as returned from application facade API.

          Basic properties do not include properties added by interceptors specific for outbound channel like metaData.http or metaData.general.locale
