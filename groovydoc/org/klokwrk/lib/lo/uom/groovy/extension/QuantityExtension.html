          one can simply use 
           quantity.format()

        • static tech.units.indriya.ComparableQuantity<Temperature> getDegC(Number num)

          Convenient creation of Quantity<Temperature> with celsius units.

          Instead of 

           Quantities.getQuantity(10, Units.CELSIUS)
          one can simply use 
           10.degC
          degC stands for celsius degree.

          Returned quantity is an instance of ComparableQuantity<Mass>.
