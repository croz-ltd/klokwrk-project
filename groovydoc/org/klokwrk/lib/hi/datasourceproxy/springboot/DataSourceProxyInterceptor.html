Package: org.klokwrk.lib.hi.datasourceproxy.springboot

[Groovy] Class DataSourceProxyInterceptor

    • org.klokwrk.lib.hi.datasourceproxy.springboot.DataSourceProxyInterceptor
    • All Implemented Interfaces and Traits:
      MethodInterceptor
      @CompileStatic
class DataSourceProxyInterceptor
extends Object
implements MethodInterceptor

      AOP alliance proxy interceptor that decorates all invocations of DataSource instance methods with datasourceproxy features.

      Some aspects of datasourceproxy features can be configured via DataSourceProxyConfigurationProperties.

      Integration with Spring Boot application is implemented via DataSourceProxyBeanPostProcessor

      See Also:
