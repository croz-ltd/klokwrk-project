          Constructs GradleDownloader with two http client dependencies.

          Http client streamingHttpClient is used for downloading potentially large Gradle distribution files. Http client headOnlyHttpClient is used only for executing HEAD requests used for getting an info about the size of there resource to be downloaded with streamingHttpClient.
